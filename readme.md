# Справочная документация

## Git

### Клонировать репозиторий:
```bash
git clone <URL>
```

### Учесть все файлы:
```bash
git add .
```

### Создание коммита:
```bash
git commit -m "..."
```

### Отправка на сервер:
```bash
git push
```

### Получить изменения с сервера:
```bash
git pull
```
